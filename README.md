### Maze generator

#### Use Randomized depth-first search to create a maze
A maze is a kind of games where a player moves in pathways with many branches to find a way
out/certain targets.

#### Start game:
Clone repository
```bash
git clone https://github.com/ngnhtrg/MazeGenerator.git
```
Run
```bash
cd MazeGenerator/
python3 main.py
```
